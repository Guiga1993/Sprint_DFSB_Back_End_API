# Sprint_DFSB_Back_End_API
This repo will contain the back_end API code reagarding the MVP requested by the discipline.

Minimal Express API

Setup:
1. npm install
2. npm run dev   # for development (requires nodemon) or npm start

Endpoints:
GET /health
GET /api/users
GET /api/users/:id
POST /api/users       { "name": "...", "email": "..." }
PUT /api/users/:id
DELETE /api/users/:id
